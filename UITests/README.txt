Project Title:
UITests (Web UI test automation)

Concepts Included:
Page Object Model
Behaviour driven development

Tools:
Maven
Cucumber-JVM
JUnit
Selenium Webdriver
Hamcrest

Requirements:
Chrome & ChromeDriver
Java
Eclipse
Firefox 

Usage:
1.Download the zip file or clone git repository.
2.Open Eclipse. File->Import->Existing Maven project->Navigate to the unzipped folder location-> Select (UITests) project.
3.Navigate to src/test/java/cucumberOptions/TestRunner.java file.
4.Right click & run as JUnit Test.

Reporting:
HTML report(open with firefox) can be found in target folder in the project(\<eclipseworkspace>\UITests\target\cucumber-reports).
